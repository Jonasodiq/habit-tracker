# Habit Tracker - Examensarbete

En mobilapplikation för att skapa, följa och analysera dagliga vanor med AI-genererade insikter.

## Projektöversikt

**Utvecklare:** Jonas N  
**Tidsplan:** 6 veckor  
**Tekniker:** React Native (Expo), AWS Serverless (Lambda, DynamoDB, Cognito)

## Funktioner (MVP)

- ✅ Användarautentisering (registrering och inloggning)
- ✅ Skapa och redigera vanor
- ✅ Markera vanor som genomförda
- ✅ Visa statistik (streaks, procent)
- ✅ AI-genererade insikter baserat på beteenden

## Projektstruktur

- `frontend/` - React Native mobilapp
- `backend/` - AWS Lambda functions och serverless config
- `docs/` - Projektdokumentation

## Komma igång

### Frontend
\`\`\`bash
cd frontend
npm install
npm start
\`\`\`

### Backend
\`\`\`bash
cd backend
npm install
serverless deploy
\`\`\`

## Utvecklingsplan

**Vecka 1:** Projektsetup och grundstruktur  
**Vecka 2:** Inloggning och användarhantering  
**Vecka 3:** Skapa vanor och databasmodell  
**Vecka 4:** Markera genomförda och statistik  
**Vecka 5:** AI-insikter  
**Vecka 6:** Testning, UI-förbättringar, dokumentation
