## ✨ Features

### MVP (Implemented)
- ✅ **Account & Authentication** — Sign up and login via AWS Cognito
- ✅ **Create & Edit Habits** — With icon, color, and frequency (daily/weekly/monthly)
- ✅ **Mark as Completed** — Toggle with animation and streak tracking
- ✅ **Statistics** — Streaks, completion rate, bar charts, calendar view
- ✅ **AI Insights** — Personalized insights generated by Claude AI
- ✅ **Tips for Weakest Habit** — AI-generated improvement tips
- ✅ **Profile** — Edit name and view account info

### Additional Features
- 🎨 Modern cyan/blue color theme
- 📅 Calendar view with daily progress rings
- 🔥 Per-habit streak tracking
- 💡 Detailed habit view (30-day data)
- 🌍 Swedish calendar support
- ⚡ Global state with 30-second cache
- 🤖 Claude AI via AWS Lambda (not directly from frontend)
- 🔒 Secure API key handling via environment variables

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- Expo Go-appen (iOS/Android)
- AWS-konto (för backend)

### 1. Clone the repository
```bash
git clone https://github.com/Jonasodiq/habit-tracker.git
cd habit-tracker
```

### 2. Backend
```bash
cd backend
npm install

# Create .env
cp .env

# Fill in
 AWS_REGION=
 COGNITO_CLIENT_ID= 
 COGNITO_USER_POOL_ID=
 ANTHROPIC_API_KEY=

 USERS_TABLE=
 HABITS_TABLE=
 COMPLETIONS_TABLE=

# Deploy to AWS
serverless deploy
```

### 3. Frontend
```bash
cd frontend
npm install

# Create .env
cp .env

# Fill in
 EXPO_PUBLIC_COGNITO_USER_POOL_ID=
 EXPO_PUBLIC_COGNITO_CLIENT_ID=
 EXPO_PUBLIC_API_BASE_URL=

# Start
npx expo start

# Test account: 
email: test9@test.com
password: Test&123456
```

## 📊 API Endpoints

| Method | Endpoint | Description |
|--------|----------|------------|
| POST | /auth/users | Register |
| POST | /auth/login | Login |
| GET | /auth/users/me | Get profile |
| PATCH | /auth/users/me | Update profile |
| GET | /habits | Get habits |
| POST | /habits | Create habit |
| PATCH | /habits/{id} | Update habit |
| DELETE | /habits/{id} | Delete habit |
| GET | /completions | Get completions |
| POST | /completions | Mark complete |
| DELETE | /completions/{id} | Unmark |
| GET | /statistics | Get statistics |
| GET | /insights | AI insights |
| POST | /insights/tips | Habit tips |
